Recipe Organizer

A simple and efficient web application 📝🍳 designed for temporarily organizing and storing recipes. Users can easily input recipe details like name, ingredients, and cooking instructions through an intuitive form 📋💡. With the "Add Item" feature, multiple recipes can be saved in a session, perfect for jotting down ideas on the go ⏳🍲. Each recipe includes a "Delete" button 🗑️✨ for easy management, and the data resets after the session ends, ensuring a fresh start every time. Whether experimenting with new dishes or tracking family favorites, this Recipe Organizer makes cooking fun and organized! 🌟👩‍🍳

Tech Stack: HTML, CSS, JavaScript

Live Demo

Blog Platform

A full-stack web application ✨ that enables users to create and publish blogs 📝 with text and image uploads 📸, interact through comments 💬, and engage with posts by adding likes ❤️. Each user has a personal profile 👤 featuring their bio, profile icon, and a collection of their blogs and liked posts, making the experience more personalized 🌟. Built using HTML, CSS, JavaScript (frontend) 🎨 and Java, Spring Boot, Hibernate, MySQL (backend) 💻, the platform demonstrates skills in user authentication, database integration, and responsive design 📱, offering an engaging space for content creation and interaction.

Tech Stack:HTML, CSS, JavaScript, MySQL, Java, Spring boot, Hibernate
